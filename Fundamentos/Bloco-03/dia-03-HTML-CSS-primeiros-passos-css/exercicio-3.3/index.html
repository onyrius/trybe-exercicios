Suelen Arruda

Quem sou

Meu nome é Suelen Arruda, e, aos 39 anos de idade, descobri que amo programação! Meu primeiro contato com a tecnologia, como estudante, se deu com a Trybe! Foi através dela que consegui ter emponderamento para seguir esse caminho!
E, depois de tanto tempo tentando achar a profissão que me apaixonasse, encontrei nos desafios das linhas de código no processo seletivo da Turma XP inc.
Hoje sou uma pessoa mais feliz e disposta a compartilhar essa felicidade com mais pessoas, sejam elas jurídicas ou físicas!
A tecnologia, a educação e arte são capazes de mudar as pessoas e, por consequência, o mundo!
